Base Theme inspired by the TALLstack but for Drupal.

Comes with :
- TailwindCSS
- AlpinJS


## Set up
Prerequisite : `rename` (`brew install rename` // `sudo apt-get install rename`)

Replace `themeName` by the name of the theme you want to setup.

```bash
sh setup.sh [themeName]
```
